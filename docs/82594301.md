::: tip

​	用通配符来匹配路由表之外的路径，进行404页面匹配

:::

```js
// router.js
import 404 from '/pages/404.vue'

const routes = [
    {
        path:'*',
        component:404
    }
]
```

::: tip

​	也可匹配指定路径下的404页面

:::

```js
// router.js
import 404 from '/pages/404.vue'

const routes = [
    {
        path:'/home-*',
        component:404
    }
]
```
